# About

In this project, I developed a website that serves as an online marketplace. I developed the back-end (using Flask library and Python) to implement various features such as login and sign-up functionality, search, automated email notifications, buy/sell/upload products and price negotiation. I leveraged HTML and CSS for the front-end interface. I also created proper project documentation like Software Requirements Specification (SRS) Document, Test Plan Document, Class Diagram and Use-Case Diagram.
